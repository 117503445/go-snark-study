# circuit vim syntax

## Installation in vim/nvim using plug
Using [Plug](https://github.com/junegunn/vim-plug), add this lines into the `.vimrc`/`init.vim`:
```
Plug 'arnaucube/go-snark-study'
Plug 'arnaucube/go-snark-study', {'rtp': 'vim-syntax'}
```
